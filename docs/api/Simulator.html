new Simulator(skillId, options)

Simulator uses the SMAPI APIs to run simulations against a skill.
Parameters:
Name Type Description
skillId String The ID of the skill to simulate against
options Object.<String, Object> The global options for the execution of the simulation that will be applied to every simulation (unless overridden in the specific call)
Properties
Name Type Attributes Default Description
locale String <optional>
 en-US The locale to use for the skill
poolingInterval Integer <optional>
 1000 The interval, in milliseconds, to poll for a simulation completion.
endpointRegion String <optional>
 NA The endpoint region of the skill
profile Profile <optional>
 The ASK CLI profile to use when calling the SMAPI SimulateSkill API
